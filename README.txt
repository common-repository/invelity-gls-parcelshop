== Description ==
Plugin Invelity GLS ParcelShop je určený pre pridanie dopravnej metódy a pop-up okna s mapou pobočiek GLS ParcelShopov na Slovensku.
Plugin pridáva doručovaciu adresu podľa vybranej pobočky.
Plugin jednoducho stiahnite, nainštalujte, aktivujte  a pridajte GLS ParcelShop v dopravných zónach.


== Installation ==

This section describes how to install the plugin and get it working.

1. Download the plugin and upload files directly via FTP (`/wp-content/plugins/invelity-ikros-invoices).
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add GLS Parcel Shop as shipping method
